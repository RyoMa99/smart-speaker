# play sound

## 概要
 pythonを用いてwavファイルを再生する。
 open-jtalkを用いて日本語のwavファイルを生成する。

## 環境
- wsl2 on ubuntu20.04
- python 3.10.2

## 環境構築
1. 参考サイトを元にWSL2でオーディオを再生するための環境を作る
2. open-jtalkの参考サイトを元にopen-jtalkに必要なライブラリ等のインストール
